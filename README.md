# Northwind on Rails

Northwind based application with Rails backend and iOS and Android client applications.

## Web

Ruby 3.3.4
Rails 7.2

```bash
git clone https://github.com/matthewblott/northwind-on-rails

cd northwind-on-rails

bundle

rails db:migrate
rails db:seed

rails server

```
